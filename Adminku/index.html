Informasi & Bantuan

Privasi Terjamin (Lokal)

AdminKu adalah layanan offline-first. Data Anda dienkripsi secara aman dan tidak pernah dikirimkan ke server internet manapun.

Pencadangan Mandiri

Gunakan fitur Ekspor (Ikon Unduh) secara berkala. Jika Anda membersihkan data cache peramban (browser), dokumen di aplikasi ini akan ikut terhapus permanen.
